# План и карта документации

Этот vault фиксирует обучение по курсу:
**"PRISMA с НУЛЯ за 70 минут — ПОЛНЫЙ КУРС (TypeScript + PostgreSQL)"**.

## Как работаем

- Каждый шаг в коде = отдельный git-коммит.
- Каждый коммит = отдельный файл в `commits/`.
- В каждом файле шага обязательно есть раздел `Команды терминала` с код-блоком `bash`.
- Формат описания свободный, ближе к учебному конспекту: без жестких секций `Плюсы/Минусы`.
- Здесь хранится краткая карта прогресса и ссылки на подробные разборы.
